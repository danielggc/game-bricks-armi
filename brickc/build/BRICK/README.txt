BrickC - Motor de Juegos de Ladrillos v1.0.0
============================================

COMO USAR:
---------
./brickc --help                    (mostrar ayuda)
./brickc --version                 (mostrar version)
./brickc examples/snake.brick      (analizar Snake)
./brickc examples/tetris.brick     (analizar Tetris)
./brickc --verbose examples/snake.brick    (modo detallado)
./brickc --json examples/snake.brick       (salida JSON)

EJEMPLOS INCLUIDOS:
------------------
examples/snake.brick    - Juego completo de Snake
examples/tetris.brick   - Juego completo de Tetris

El analizador valida la sintaxis del lenguaje BrickLang
y construye la tabla de símbolos con todas las entidades,
reglas y configuraciones del juego.

Compatible con Windows XP - Tamaño total: ~400KB
